元素节点对象（element）​

  • 在网页中，每一个标签都是一个元素节点

  • 如何获取元素节点对象?

    1. 通过document来获取对象
    2. 通过document来创建对象

  • 通过document来获取已有对象

    1. document.getElementById()
    • 根据id来获取一个元素节点
    1. ducument.getElementsTagName()
    • 根据标签名来获取一组元素节点
    • 返回的结果是可以实时更新的集合
      • document.getElementsByTagName("*") 获取页面中所有的元素
    1. document.getElementsByClassName()
    • 根据类名来获取一组元素节点
    • 返回的是一个类数组对象
    • 该方法返回的结果是一个实时更新的集合
      • 当网页中新添加元素时，集合也会实时的刷新
    1. document.getElementsByName()
    • 根据name属性来获取一组元素节点
    • 返回一个实时更新的集合
      • 主要用于表单项
    1. document.querySelectorAll()
    • 根据选择器去页面中查询一组元素
      • 会返回一个类数组（不会实时更新）
    1. document.querySelector()
    • 根据选择器去页面中查询第一个符合条件的元素

  • 通过document来创建一个元素节点

    • document.createElement(”标签名“)

      • 根据标签名创建一个元素节点对象

元素节点​

  • div元素的原型链
    • HTMLDivElement -> HTMLElement -> Element -> Node -> ...
  • 通过元素节点获取其他节点的方法
    • element.childNodes 获取元素的所有子节点
    • element.children 获取当前元素的子元素
    • element.firstElementChild 获取第一个子元素
    • element.lastElementChild 获取最后一个子元素
    • element.nextElementSibling 获取下一个兄弟元素
    • element.prevElementSibling 获取前一个兄弟元素
    • element.parentNode 获取当前元素的父节点
    • element.tagName 获取当前元素的标签名

文本节点​

在DOM中，网页中所有的文本内容瓯都市文本节点对象， -可以通过元素来获取其中的文本节点对象，但是我们通常不会这么做

我们可以通过元素去修改其中的文本

  • 修改文本的三个属性
    • element.textContent 获取或修改元素中的文本内容
      • 获取的是标签中的内容，不会考虑CSS样式
    • element.innerText 获取或修改元素中的文本内容
      • innerText获取内容时，会考虑CSS样式
      • 通过innerText去读取CSS样式,会触发网页的重排（计算CSS样式）
      • 当字符串中有标签时，会自动对标签转义
      //<li> ---> $lt;li$gt;
  
- element.innerHTML 获取或修改元素中html代码
  - 可以直接向元素中添加html代码
  - innerHTML插入内容时，有被xss注入的风险

属性节点(Attr)​

  • 在DOM也是一个对象，通常不需要获取对象而是直接通过元素即可完成对其的读取和修改
  • 如何操作属性节点：
    • 方式一：
      • 读取：元素.属性名（注意，class属性需要使用className来读取）
        • 读取一个属性名和属性值相同的属性时，会返回true或false
      • 修改：元素.属性名 = 属性值
    • 方式二：
      • 读取：元素.getAttribute(属性名)
      • 修改：元素.setAttribute(属性名, 属性值)
      • 删除：元素.removeAttribute(属性名)
const input = document.getElementsByName("username")[0]

const input = document.querySelector("[name=username]")

console.log(input.type)

console.log(input.getAttribute("type"))

input.setAttribute("value", "孙悟空")

input.setAttribute("disabled", "disabled")

事件​

  • 事件就是用户和页面之间发生的交互行为
    • 比如：点击事件、鼠标移动、双击、键盘事件
  • 可以通过为事件绑定响应函数（回调函数），来完成和用户之间的交互
  • 绑定响应函数的方式：
    1. 可以直接在元素属性中绑定事件
    2. 可以通过元素的至指定属性设置回调函数的形式绑定事件（一个事件只能绑定一个响应函数）
    3. 可以通过元素addElementListener()方法来绑定事件
<button id="btn" onmouseenter="alert('鼠标移入事件触发')">点我一下</button>

const btn = doucument.getElementById("btn")
// 通过设置回调函数 为btn绑定事件
btn.onmouseenter = function(event){
    alear("鼠标移入了")
}


btn.addElementListener("click", (event) => {
    alear("通过addElementListener绑定事件")
    // 第三个参数设置为true 把事件的传播改为捕获阶段 
})

文档的加载​

  • 页面加载是自上向下加载的，如果将js代码编写到网页的上边，js代码在执行时，网页还没加载完毕，这时会出现无法获取到DOM对象的情况

    • windon.onload 事件会在窗口中得内容加载完毕之后才触发

    • document得DOMCibtebtLoaded事件会在当前文档加载完毕之后触发

    • 如何解决这个问题：

      1. 将script标签编写到body的最后
      2. 将代码编写到window.onload的回调函数中
      3. 将代码编写到document对象DOMContentLoaded的回调函数中（执行时机更早）
      4. 将代码编写到外部的js文件中，然后defer的形式引入(执行时机更早，早于DOMContentLoaded)

节点的复制​

  • 使用cloneNode()方法对节点进行复制时，他会复制节点的所有特点包括各种属性。
    • 这个方法默认只会复制当前节点，二不会复制节点的子节点
    • 可以传递一个true作为参数，这样该方法也会将元素的子节点一起复制 // 浅复制
    • const newNode = OldNode.CloneNode() // 深复制
    • const newNode = OldNode.CloneNode(true)

修改css样式​

  • 修改样式的方式：
    • 元素.style.样式名 = 样式值
    • 样式值需要加单位
    • 如有样式含有 - 则需要将样式表修改为驼峰命名法

读取css样式​

getComputedStyle()

  • 他返回一个对象，这个对象中包含了当前元素所有的生效样式
  • 参数：
    1. 要获取样式的对象
    2. 要获取的伪元素
  • 返回值：
    • 返回一个对象，对象中存储了当前元素的样式
  • 注意：
    • 样式对象中返回的样式值，不一定能拿来直接计算
      • 使用时，一定要确保是可以计算的再去计算
const styleObj = getComputedStyle(元素对象)

console.log(styleObj.width) // styleObj中有所有当前元素对象生效的css样式

box1.style.width = parseInt(styleObj.width) + 100 + "px"

const beforeStyle = getComputedStyle(box1, "::before") // 第二个参数用来获取伪元素中的样式

读取样式的方式​

  • 元素.clientHeight

  • 元素.clientWitdth

    • 获取元素内部的宽度和高度（包括内容区和内边距）

  • 元素.offsetHeight

  • 元素.offsetWidth

    • 获取元素的可见框大小 （包括内容区、内边距和边框）

  • 元素.scrollHeight

  • 元素.scrollWidth

    • 获取元素滚动区域的大小

  • 元素.offsetParent

    • 获取元素的定位父元素
    • 定位父元素：离当前元素最近的开启了定位的祖先元素，
      • 如果所有的元素都没有开启定位则返回body

  • 元素.offsetTop

  • 元素.offsetLeft

    • 获取元素相对于其定位父元素的偏移量

  • 元素.scrollTop

  • 元素.scrollLeft

    • 获取或设置元素滚动条的偏移量

通过class修改css样式​

  • 除直接修改样式外，也可以通过修改class属性来间接的修改样式

    • 通过class修改样式的好处：
      1. 可以一次修改多个样式
      2. 对JS和CSS进行解耦

  • 方式一：

    • 元素.className += "box2" (不推荐)

  • 方式二：

  • 元素.classList 是一个对象，对象中提供了对当前元素的类的各种操作方法

    • 元素.classList.add(...className) 向元素中添加一个或多个class
    • 元素.classList.remove() 删除元素中的一个或者多个class
    • 元素.classList.replace() 替换元素中的class
    • 元素.classList.toggle() 对class进行切换
    • 元素.classList.contains() 检查是否含有该class 返回布尔值
